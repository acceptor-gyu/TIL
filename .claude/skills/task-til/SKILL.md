# 새로운 TIL 항목 추가

이 skill은 새로운 학습 항목을 프로젝트에 추가합니다.

## 수행 작업

1. **README 업데이트**: 날짜별 최신순, 같은 날이라면 생성 번호 내림차순으로 해당 카테고리에 새 항목 추가
2. **내용 작성**: 양식에 맞는 또는 관련 내용 중 추가로 알았으면 좋은 내용을 공식 문서를 참고해서 작성
3. **키워드 설명**: 선정되어있는 키워드에 대한 설명 작성

## 사용법

```
/task-til 파일이름 또는 /task-til @파일이름
```

**예시:**
- `/task-til 250203_01_Spring_Layered_Architecture_Test.md`
- `/task-til 260216_03_토큰_최적화_활용_방법.md`
- `/task-til 260216_02_수강신청_시스템_초당_10000건_요청_처리_아키텍처.md`

## 인자 처리

- **$ARGUMENTS[0]** 또는 **$0**: 파일명 (예: 260216_03_토큰_최적화_활용_방법.md)

## 주의사항

- 카테고리가 README에 없으면 새로 생성
- 파일명은 자동으로 스네이크 케이스로 변환 (공백 → 언더스코어)
- README의 항목은 날짜 역순으로 정렬 (최신이 위)
- 날짜 형식: YYYY.MM.DD
- 참고 문서는 가능하면 공식문서를 참고하도록

## 실행 후 확인

1. README.md에 항목이 추가되었는지 확인
2. 사용자에게 생성된 파일 경로와 다음 단계 안내
