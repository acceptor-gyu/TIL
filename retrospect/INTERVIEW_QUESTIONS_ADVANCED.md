## 🎯 사용 방법

### 면접관용
1. 각 섹션별 파일을 열어 질문 선택
2. 후보자 수준에 따라 난이도 조절
3. 답변 토글을 펼쳐 모범 답안 확인
4. 평가 기준에 따라 점수 산정

### 후보자용
1. 각 질문을 먼저 스스로 답변해보기
2. 토글을 열어 모범 답안과 비교
3. 부족한 부분 학습 자료로 보완
4. 실시간 코딩 챌린지로 실전 연습

## 🔥 핵심 평가 영역

### 기술 깊이 (30점)
- JPA, Spring, 동시성, 데이터베이스 내부 동작 원리
- 표면적 지식이 아닌 "왜" 그런지 설명 능력

### 문제 해결 (25점)
- 논리적 사고, 트레이드오프 판단
- 재난 시나리오 대응, 디버깅 능력

### 코드 품질 (20점)
- 클린 코드, 설계 원칙, 리팩토링
- 실제 코딩 챌린지 수행 능력

### AI 활용 (15점)
- 프롬프트 엔지니어링, 비판적 검토
- AI를 도구로 활용하는 능력

### 커뮤니케이션 (10점)
- 설명 명확성, 논리 전개
- 압박 상황 대처 능력
